Building a Critical Mass Sound System - Part 2

Remote sounds

I’m going to assume you have some basic technical skill and will be able to install the Pi operating system, connect your monitor, mouse and keyboard without my help.

Parts List

  • Raspberry Pi (Version B+) (£30): This is a small computer that will automatically start our bluetooth network. Raspberry Pi B+ on Amazon.

  • Raspberry Pi Case (£6): You can choose whatever you like, or entirely skip it. I chose the fairly robust ModMyPi RPi Model B+ Case.

  • Power supply: (£5) You’ll need a power supply as one isn’t provided. I can’t remember where I bought mine, but any micro USB charger should do.

  • NOOBS SD Card (£7): I’d recommend the NOOBS SD Card.

  • Bluetooth Adapter (£10): ASUS products are well regarded, compatible with the Raspberry Pi and are not too expensive. The Asus USB-BT400 fit the bill.

  • HDMI to DVI Cable (£5): This is needed so we can connect the monitor to the Pi and see what we’re doing. I went ahead and bought the World of Data - 1m HDMI to DVI Cable .

Total: £64 - Not too shabby for a system that you can remotely send audio to.

Operating System

I used Arch Linux from the NOOBs setup. This operating system doesn’t come with a user interface so having some experience with the terminal is required.
